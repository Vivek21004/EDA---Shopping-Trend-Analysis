Project Description: Exploratory Data Analysis of Shopping Trends

This project focuses on performing Exploratory Data Analysis (EDA) on a customer shopping trends dataset to identify patterns in purchasing behavior, customer demographics, and product preferences. The dataset contains 3,900 records and 18 variables, including information about customers such as age, gender, purchased items, product categories, purchase amount, location, payment methods, and promotional usage.

The analysis was conducted using Python with libraries such as Pandas, NumPy, Matplotlib, Seaborn, and Plotly for data manipulation, visualization, and insight generation.

The project begins with data exploration and preprocessing, where the dataset is loaded, column information is examined, and unique values in key variables are analyzed. Since the dataset contains no missing values, the analysis proceeds directly to exploring relationships between variables.

Several analytical questions are addressed in the project, including:

Understanding the distribution of customer ages

Analyzing average purchase amounts across product categories

Identifying gender-based purchasing patterns

Examining seasonal shopping trends

Evaluating the impact of discounts and promo codes on purchases

Studying customer loyalty through subscription status and previous purchases

Identifying preferred payment methods and shipping types

Through data visualization techniques such as histograms, bar charts, and interactive plots, the project uncovers patterns in consumer behavior and highlights factors that influence purchasing decisions.

The insights generated from this analysis can help businesses improve marketing strategies, optimize product offerings, and enhance customer experience by understanding customer preferences and shopping habits.
